# TradingView Fibonacci Trading Bot

Автоматический индикатор для торговли криптовалютами на основе уровней Fibonacci.

## Структура проекта

```
TradingView/
├── indicators/
│   └── fibonacci_indicator.pine    # Основной индикатор Pine Script
├── docs/
│   └── strategy_logic.md          # Описание логики стратегии
└── README.md                       # Этот файл
```

## Этап 1 - Базовый индикатор

### Функционал:
- ✅ Построение Fibonacci от максимума к минимуму
- ✅ Поиск минимума за N дней (настраиваемо)
- ✅ Проверка роста ≥100% от минимума
- ✅ Проверка "пламени свечи" >15%
- ✅ Сигналы входа на уровнях 0.618, 0.5, 0.382
- ✅ Сигналы выхода на соответствующих целевых уровнях

### Установка:
1. Открыть TradingView
2. Перейти в Pine Editor (внизу графика)
3. Скопировать код из `indicators/fibonacci_indicator.pine`
4. Нажать "Add to Chart"

### Настройки индикатора:
- **Lookback Period**: период поиска минимума (по умолчанию 150)
- **Min Growth %**: минимальный рост для построения Fibonacci (по умолчанию 100%)
- **Wick Threshold %**: порог пламени свечи (по умолчанию 15%)

## Использование:
1. График обязательно в **логарифмическом масштабе**
2. Таймфрейм: **1D (дневной)**
3. Индикатор автоматически построит Fibonacci при выполнении условий
4. Зеленые стрелки = сигналы покупки
5. Красные стрелки = сигналы продажи
